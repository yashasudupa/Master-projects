Detailed Description

template<class S>
class otawa::sem::StateIter< S >

This class, based on BBIter, allows to traverse all semantic execution paths of a basic block while maintaining the consistency of the analysis state.

The parameter class must be implement the StateManager concept described below:

class StateManager {
public:
typedef ... t; // type of states
t bot(void); // bottom least value
void free(t s); // free a state
bool equals(t s1, t s2); // test for equality
t copy(t s); // provide a fresh copy of state (fried by free())
t join(t s1, t s2); // join both states
};

And type StateManager::t must support the assignment operation.

Parameters
SState manager (must implement StateManager concept above).
