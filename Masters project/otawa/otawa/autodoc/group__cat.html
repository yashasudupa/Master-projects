Detailed Description

Implementation of LRU instruction cache analysis by categorisation according to the paper:

C. A.Healy, R. D. Arnold, F. Mueller, D. B. Whalley, M. G. Harmon Bounding Pipeline and Instruction Cache Performance. IEEE Transactions on Computers, 1/1999.
