Detailed Description

This module provides classes to perform static analysis and Data Flow Analyses (DFA).

This includes DFA engines:

  • otawa::dfa::IterativeDFA,
  • otawa::dfa::XIterativeDFA,
  • otawa::util::HalfAbsInt.

This includes also tool classes:

  • otawa::dfa::BitSet to represents efficiently sets.
