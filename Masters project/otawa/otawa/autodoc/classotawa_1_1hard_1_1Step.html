Detailed Description

An hardware step is a step or an action to perform inside the microprocessor to execute an instruction.

Each step may have its own parameter. To get the steps to execute an instruction, one has to use the method Processor::execute() and pass the instruction and vector to store the steps in.

Allowed steps follow (notice that all action are done at the start of the cycle):

  • NONE (no parameter) – invalid step (only used for convenience).
  • STAGE (Stage) – stage to enter.
  • FU (FunctionalUnit) – functional to be enter.
  • READ (Register) – register to read.
  • WRITE (Register) – register to write.
  • USE (Queue) – queue entry to allocate.
  • RELEASE (Qeueu) – queue entry to release.
