Detailed Description

This class is an abstract class to monitor the load of flow facts.

To get simple access to the flow fact files, a new class must be inherited from this class and this class must implement the virtual "onXXX" methods.

Required Features
  • dfa::INITIAL_STATE_FEATURE
Provided features
  • otawa::FLOW_FACTS_FEATURE
  • otawa::MKFF_PRESERVATION_FEATURE
Configuration
  • otawa::FLOW_FACTS_PATH – select the flow fact to load (if not present, the flow fact name is obtained by replacing extension or by appending ".ff" or ".ffx").
  • otawa::FLOW_FACTS_MANDATORY – if set to true, the processing fails if one loop bound is not available (default to false).
See also
F4 : Flow Facts File Format for more details on the flow facts files.
Author
H. Cass casse.nosp@m.@iri.nosp@m.t.fr
