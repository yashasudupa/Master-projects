otawa::dfa::Value::Value ( t::uint32  base,
t::uint32  delta,
t::uint32  count 
)

According to the triple (base, delta, count), build a value as:

  • a constant (if count = 0),
  • an interval (if delta = 1),
  • a CLP, i.e., a value in { base + delta * k / 0 <= k <= count }
